How to Run

  1. Download FlappyBirds.jar from the latest release.
  2. Make sure Java is installed (JDK/JRE).
  3. Run: java -jar FlappyBirds.jar

Upload a New Release

  1. Create a new GitHub Release (tag like v1.2.3).
  2. Attach your built .jar (and/or a .zip).
  3. This website updates automatically to point to that release.
